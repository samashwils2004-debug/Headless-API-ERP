# AdmitFlow Monorepo

## Apps
- `apps/api`: FastAPI runtime and infrastructure engines
- `apps/web`: Next.js landing, docs, and console

## Packages
- `packages/templates`: workflow template JSON definitions
- `packages/blueprint-schema`: JSON schemas for blueprint and workflow validation
- `packages/sdk`: reserved for future SDK

## Quick Start
1. `docker-compose up`
2. `npm run web:dev`
