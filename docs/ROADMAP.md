# AdmitFlow Roadmap

## Phase 1: Runtime and API Foundations
- Deterministic workflow engine and safe condition parser
- Event persistence and stream fanout
- Multi-tenant enforcement across API and DB paths

## Phase 2: Console and AI Compile UX
- Control-plane screens for projects, workflows, AI generator, events
- 4-stage validation visibility and deploy guardrails

## Phase 3: Production Hardening
- Observability baseline (Sentry, Prometheus)
- Load and failure-mode testing
- Security checklist closure

## Phase 4: Ecosystem
- Shared template package growth
- External SDK and integration tooling
